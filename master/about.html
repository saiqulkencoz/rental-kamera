About

Juragan Kamera Web merupakan sebuah website yang kami buat untuk membantu para calon pengguna jasa rental kamera mudah dan cepat untuk kebutuhan ringan maupun profesional.

Sebelum tahu tentang siapa yang mengelola Juragan Kamera Web saya ingin bercerita tentang berdirinya jasa rental kamera, Juragan Kamera terlebih dulu.

Website ini dibuat atas dasar untuk memenuhi final project pada mata kuliah Pengenalan Perancangan Web (PPW). Seiring berjalannya waktu, kami sadar bahwa website ini memang harus ada dalam bisnis yang teman saya "abi" geluti. Akhirnya website ini terbuat dan sudah mengudara, dan kami bertiga pun menjadi aktor dibelakangnya.
